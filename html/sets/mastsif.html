The CUTEr/st Test Problem Set

The Constrained and Unconstrained Testing Environment, revisited/safe threads, (CUTEr/ CUTEst) contains a set of test problems for linear and nonlinear optimization.

The complete set

The test problem set is available using svn via svn checkout --username anonymous http://ccpforge.cse.rl.ac.uk/svn/cutest/sif/trunk ./sif

(with a blank password) or as as a pair of compressed (gzipped) tar files. For the sake of efficiency, those problems whose SIF files are less than 100 Kbytes are available in one tar file (roughly 1.2 Mbytes in total), while the remaining set of larger problems are contained in another (approximately 44.2 Mbytes in total). A list of newly-added problems is available.

Test problem subsets

As each test problem is classified as to what sort of objective and constraints are involved, it is possible to select subsets of the test problems which conform to predefined criteria. An unsophisticated version of software to do this is distributed with CUTEr/CUTEst.
