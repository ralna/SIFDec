4.4 Data Cards

The ELEMENTS and ENDATA indicator cards perform the same function as the cards NAME and ENDATA in Section 3.2.1 and 3.2.2. The problem name specified in field 3 on the ELEMENTS card must be the same as that given in the same field on the NAME card of the SDIF file.
