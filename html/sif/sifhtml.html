The SIF Reference Document

Andrew R. Conn, Nicholas I. M. Gould and Philippe L. Toint

Abstract:

We provide a definitive description of the standard input format (SIF). It is intended to be used primarily as a reference document.
Special syntax to cope with quadratic terms in the objective function has been added.
